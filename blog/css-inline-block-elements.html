What is the difference between Inline and Block elements in HTML?



Block level elements start on a new line.

Common examples are:

  • <div>
  • <p>
  • <h1> -<h6>

Inline elements don't start on a new line.

They take as much width as necessary where started.

Common examples are:

  • <span>
  • <img>
  • <a>
